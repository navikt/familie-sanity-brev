# Familie sanity brev

### Installere Sanity
* Kjør `npm install -g @sanity/cli` i terminalen.

### Kjøre lokalt
* Kjør `yarn` for å installere alle npm-pakkene. 
* Kjør `sanity dev` for å starte applikasjonen på localhost:3333.
